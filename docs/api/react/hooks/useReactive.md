# useReactive

提供一种数据响应式的操作体验。

## 示例

```ts
import { useReactive } from '@base-web-kits/base-tools-react';

const state = useReactive({ count: 0 });
```

## 参数

- `initialState (Object)`: 初始状态。

## 返回值

- `(Object)`: 响应式状态。
