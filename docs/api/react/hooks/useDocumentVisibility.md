# useDocumentVisibility

监听文档可见性状态。

## 示例

```ts
import { useDocumentVisibility } from '@base-web-kits/base-tools-react';

const documentVisibility = useDocumentVisibility();
```

## 返回值

- `(string)`: 'visible' | 'hidden'。
