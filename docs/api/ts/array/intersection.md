# intersection

创建一个包含所有给定数组共享元素的新数组。

## 示例

```ts
import { intersection } from '@base-web-kits/base-tools-ts';

intersection([2, 1], [2, 3]);
// 结果: [2]
```

## 参数

- `arrays (...Array)`: 要检查的数组。

## 返回值

- `(Array)`: 返回新数组。
