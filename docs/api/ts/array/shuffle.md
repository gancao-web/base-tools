# shuffle

创建一个打乱顺序的数组副本。

## 示例

```ts
import { shuffle } from '@base-web-kits/base-tools-ts';

shuffle([1, 2, 3, 4]);
// 结果: [4, 1, 3, 2] (随机)
```

## 参数

- `collection (Array)`: 要打乱的集合。

## 返回值

- `(Array)`: 返回打乱后的新数组。
