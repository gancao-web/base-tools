# upperCase

将字符串转换为大写，并以空格分隔单词。

## 示例

```ts
import { upperCase } from '@base-web-kits/base-tools-ts';

upperCase('--foo-bar--');
// 结果: 'FOO BAR'

upperCase('fooBar');
// 结果: 'FOO BAR'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回转换后的字符串。
