# pascalCase

将字符串转换为 PascalCase（大驼峰写法）。

## 示例

```ts
import { pascalCase } from '@base-web-kits/base-tools-ts';

pascalCase('Foo Bar');
// 结果: 'FooBar'

pascalCase('fooBar');
// 结果: 'FooBar'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回 PascalCase 的字符串。
