# capitalize

将字符串的首字母转换为大写，其余字符转换为小写。

## 示例

```ts
import { capitalize } from '@base-web-kits/base-tools-ts';

capitalize('FRED');
// 结果: 'Fred'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回转换后的字符串。
