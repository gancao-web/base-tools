# startCase

将字符串转换为 Start Case（每个单词首字母大写）。

## 示例

```ts
import { startCase } from '@base-web-kits/base-tools-ts';

startCase('--foo-bar--');
// 结果: 'Foo Bar'

startCase('fooBar');
// 结果: 'Foo Bar'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回 Start Case 的字符串。
