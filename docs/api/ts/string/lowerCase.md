# lowerCase

将字符串转换为小写，并以空格分隔单词。

## 示例

```ts
import { lowerCase } from '@base-web-kits/base-tools-ts';

lowerCase('--Foo-Bar--');
// 结果: 'foo bar'

lowerCase('fooBar');
// 结果: 'foo bar'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回转换后的字符串。
