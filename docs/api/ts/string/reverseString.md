# reverseString

反转字符串。

## 示例

```ts
import { reverseString } from '@base-web-kits/base-tools-ts';

reverseString('abc');
// 结果: 'cba'
```

## 参数

- `string (string)`: 要反转的字符串。

## 返回值

- `(string)`: 返回反转后的字符串。
