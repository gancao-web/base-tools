# upperFirst

将字符串的首字母转换为大写。

## 示例

```ts
import { upperFirst } from '@base-web-kits/base-tools-ts';

upperFirst('fred');
// 结果: 'Fred'

upperFirst('FRED');
// 结果: 'FRED'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回转换后的字符串。
