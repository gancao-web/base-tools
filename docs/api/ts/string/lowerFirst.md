# lowerFirst

将字符串的首字母转换为小写。

## 示例

```ts
import { lowerFirst } from '@base-web-kits/base-tools-ts';

lowerFirst('Fred');
// 结果: 'fred'

lowerFirst('FRED');
// 结果: 'fRED'
```

## 参数

- `string (string)`: 要转换的字符串。

## 返回值

- `(string)`: 返回转换后的字符串。
