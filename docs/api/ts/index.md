**安装**

`npm i @base-web-kits/base-tools-ts`

**使用**

```ts
import { toDayjs, getUrlParam, toMaskPhone } from '@base-web-kits/base-tools-ts';
```

**es-toolkit**

内置[es-toolkit](https://es-toolkit.dev/) , 可直接使用es-toolkit的所有函数

```ts
import { cloneDeep, randomInt } from '@base-web-kits/base-tools-ts';
```
