# isSubsetWith

接受一个 `customizer` 函数来定制比较的 `isSubset`。

## 示例

```ts
import { isSubsetWith } from '@base-web-kits/base-tools-ts';

// 示例略，类似 isSubset 但带有 customizer
```

## 参数

- `object (Object)`: 要检查的对象。
- `source (Object)`: 属性值来源对象。
- `customizer (Function)`: 定制比较的函数。

## 返回值

- `(boolean)`: 如果匹配返回 `true`，否则返回 `false`。
