# isNumber

检查 `value` 是否是 `number` 类型。

## 示例

```ts
import { isNumber } from '@base-web-kits/base-tools-ts';

isNumber(1);
// 结果: true

isNumber('1');
// 结果: false
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 `number` 返回 `true`，否则返回 `false`。
