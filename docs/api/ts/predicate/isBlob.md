# isBlob

检查 `value` 是否是 Blob 对象。

## 示例

```ts
import { isBlob } from '@base-web-kits/base-tools-ts';

isBlob(new Blob());
// 结果: true
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 Blob 返回 `true`，否则返回 `false`。
