# isBoolean

检查 `value` 是否是 `boolean` 类型。

## 示例

```ts
import { isBoolean } from '@base-web-kits/base-tools-ts';

isBoolean(true);
// 结果: true

isBoolean(null);
// 结果: false
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 `boolean` 返回 `true`，否则返回 `false`。
