# isUndefined

检查 `value` 是否是 `undefined`。

## 示例

```ts
import { isUndefined } from '@base-web-kits/base-tools-ts';

isUndefined(undefined);
// 结果: true

isUndefined(null);
// 结果: false
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 `undefined` 返回 `true`，否则返回 `false`。
