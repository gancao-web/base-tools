# isNull

检查 `value` 是否是 `null`。

## 示例

```ts
import { isNull } from '@base-web-kits/base-tools-ts';

isNull(null);
// 结果: true

isNull(undefined);
// 结果: false
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 `null` 返回 `true`，否则返回 `false`。
