# isJSONArray

检查 `value` 是否是 JSON 数组。

## 示例

```ts
import { isJSONArray } from '@base-web-kits/base-tools-ts';

isJSONArray([1, 2]);
// 结果: true
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 JSON 数组返回 `true`，否则返回 `false`。
