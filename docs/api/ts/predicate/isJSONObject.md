# isJSONObject

检查 `value` 是否是 JSON 对象。

## 示例

```ts
import { isJSONObject } from '@base-web-kits/base-tools-ts';

isJSONObject({ a: 1 });
// 结果: true
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 JSON 对象返回 `true`，否则返回 `false`。
