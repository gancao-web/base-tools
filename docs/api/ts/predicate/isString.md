# isString

检查 `value` 是否是 `string` 类型。

## 示例

```ts
import { isString } from '@base-web-kits/base-tools-ts';

isString('abc');
// 结果: true

isString(1);
// 结果: false
```

## 参数

- `value (any)`: 要检查的值。

## 返回值

- `(boolean)`: 如果是 `string` 返回 `true`，否则返回 `false`。
