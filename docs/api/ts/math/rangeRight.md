# rangeRight

创建一个包含从 `start` 到 `end`，但不包含 `end` 的数字数组，顺序相反。

## 示例

```ts
import { rangeRight } from '@base-web-kits/base-tools-ts';

rangeRight(4);
// 结果: [3, 2, 1, 0]
```

## 参数

- `start (number)`: 开始范围。
- `end (number)`: 结束范围。
- `step (number)`: 步长。

## 返回值

- `(number[])`: 返回数字数组。
