# median

计算数组的中位数。

## 示例

```ts
import { median } from '@base-web-kits/base-tools-ts';

median([1, 2, 3]);
// 结果: 2

median([1, 2, 3, 4]);
// 结果: 2.5
```

## 参数

- `array (number[])`: 要计算的数组。

## 返回值

- `(number)`: 返回中位数。
