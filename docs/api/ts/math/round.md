# round

根据精度四舍五入数字。

## 示例

```ts
import { round } from '@base-web-kits/base-tools-ts';

round(4.006);
// 结果: 4

round(4.006, 2);
// 结果: 4.01
```

## 参数

- `number (number)`: 要四舍五入的数字。
- `precision (number)`: 精度。

## 返回值

- `(number)`: 返回四舍五入后的数字。
