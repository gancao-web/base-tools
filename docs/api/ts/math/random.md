# random

生成一个随机浮点数。

## 示例

```ts
import { random } from '@base-web-kits/base-tools-ts';

random(0, 5);
// 结果: 0 到 5 之间的随机浮点数
```

## 参数

- `lower (number)`: 下限。
- `upper (number)`: 上限。
- `floating (boolean)`: 是否返回浮点数。

## 返回值

- `(number)`: 返回随机数。
