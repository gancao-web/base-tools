# randomInt

生成一个随机整数。

## 示例

```ts
import { randomInt } from '@base-web-kits/base-tools-ts';

randomInt(0, 5);
// 结果: 0 到 5 之间的随机整数
```

## 参数

- `lower (number)`: 下限。
- `upper (number)`: 上限。

## 返回值

- `(number)`: 返回随机整数。
