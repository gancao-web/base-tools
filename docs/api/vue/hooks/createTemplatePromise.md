# createTemplatePromise

## 描述

创建一个模板 Promise，用于以 Promise 方式调用组件。

## 示例

````ts
import { createTemplatePromise } from '@base-web-kits/base-tools-vue';

const TemplatePromise = createTemplatePromise();
const result = await TemplatePromise.start();
``` 示例代码
````
