# onClickOutside

监听元素外部点击事件。

## 示例

```ts
import { onClickOutside } from '@base-web-kits/base-tools-vue';

onClickOutside(target, (event) => console.log(event));
```

## 参数

- `target (Element)`: 目标元素。
- `handler (Function)`: 回调函数。
