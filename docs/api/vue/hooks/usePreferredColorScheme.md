# usePreferredColorScheme

## 描述

响应式的 prefers-color-scheme 媒体查询。

## 示例

```ts
import { usePreferredColorScheme } from '@base-web-kits/base-tools-vue';

const colorScheme = usePreferredColorScheme(); // 'no-preference' | 'light' | 'dark'
```
