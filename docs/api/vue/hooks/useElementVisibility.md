# useElementVisibility

## 描述

检测元素是否可见。

## 示例

```ts
import { useElementVisibility } from '@base-web-kits/base-tools-vue';
import { ref } from 'vue';

const target = ref(null);
const targetIsVisible = useElementVisibility(target);
```
