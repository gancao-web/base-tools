# usePreferredReducedMotion

## 描述

响应式的 prefers-reduced-motion 媒体查询。

## 示例

```ts
import { usePreferredReducedMotion } from '@base-web-kits/base-tools-vue';

const motion = usePreferredReducedMotion(); // 'no-preference' | 'reduce'
```
