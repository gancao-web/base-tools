# usePreferredContrast

## 描述

响应式的 prefers-contrast 媒体查询。

## 示例

```ts
import { usePreferredContrast } from '@base-web-kits/base-tools-vue';

const contrast = usePreferredContrast(); // 'no-preference' | 'more' | 'less' | 'custom'
```
