# useTextareaAutosize

## 描述

自动调整 textarea 的高度。

## 示例

```ts
import { useTextareaAutosize } from '@base-web-kits/base-tools-vue';

const { textarea, input } = useTextareaAutosize();

// 绑定到 textarea 元素
// <textarea ref="textarea" v-model="input" />
```
